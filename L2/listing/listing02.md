Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main
import (
	"fmt"
)

func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}
func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}
func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
test():
В этом случае переменная х является по сути глобальной, и мы присваиваем ей значение единицы, а затем при вызове defer
инкрементируем это значение, тогда при возврате из функции инкрементируем значение глобальной переменной и получаем 2.

anotherTest():
При вызове функции anotherTest() значение х определено перед defer. Когда выполнение программы доходит до defer, то defer 
будет работать с текущим значением х, будет выведено значение 1: при инициализации х получает значение 0, после дефера - 1,
затем происходит return из функции.
```

Что выведет программа? Объяснить вывод программы.

```go
package main
func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()
	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Программа выведет все переданные в отдельной горутине значения, а затем будет дедлок,
так как при печати значений из канала в самом конце будем читать из пустово канала.
Чтобы это предотвратить, необходимо выполнить функцию закрытия канала close(ch) после передачи
всех данных в канал.
```
